## 🏗 Featured Project

### AERA – Workforce & HR Management Platform

AERA is a modern workforce management platform designed to streamline business operations and employee administration.

#### Key Features

* 🔐 Microsoft Entra ID Single Sign-On (SSO)
* 👥 Team & Role Management
* 🛡 RBAC Permission System
* 📅 Vacation & Absence Management
* ⏰ Work Time Tracking
* 🔔 Notification Platform
* 📄 Document Management
* ☁️ Azure Blob Storage Integration
* 📊 Reporting & Analytics
* 📧 Email & Microsoft Teams Integration
* 🔄 Shift Planning & Auto Assignment
* 🧪 Automated Testing & Quality Assurance

#### Technical Highlights

* Better Auth
* Microsoft Entra ID
* Prisma ORM
* PostgreSQL
* Next.js 15
* React
* TypeScript
* Azure Blob Storage
* Redis / Upstash
* Selenium E2E Testing
* Jest
* Puppeteer PDF Generation
* Role-Based Access Control
* Security Hardening
* Distributed Rate Limiting
* Idempotency Protection
